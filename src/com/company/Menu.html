Main Menu Info

From the main menu you can choose to go into service setup, or service monitoring mode. To use service monitoring mode, you will need to install Wireshark from here.

Service Setup

This mode does not automate service setup, it just provides information (available by clicking the info button or by clicking this.) about network traffic, port-forwarding, and connections, then tests the connectivity of the user’s service.

Monitoring Mode

This mode uses the service’s port and IP to generate some Wireshark filters. The user can then only show traffic to and from that service. If Wireshark is installed, you maybe able to open it from within the service setup tool. More information is available about Wireshark here
